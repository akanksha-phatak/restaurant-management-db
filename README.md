# Restaurant Management Database System

## 📌 Description
This project is a DBMS-based system designed to manage restaurant operations including customers, orders, menu, delivery, and payments.

## 🛠️ Technologies Used
- MySQL (DB Fiddle)
- SQL

## ⚙️ Features
- Customer Management
- Order Processing
- Menu Handling
- Billing System
- Delivery Tracking

## 📊 Database Components
- Customers
- Orders
- Menu
- Order_Items
- Delivery
- Payment
